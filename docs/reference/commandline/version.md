```markdown
Show the TWIC version information

Usage:
  twic version [flags]
```

## Examples

```bash
$ twic version
Version:     0.1.6
Git commit:  HEAD
Built:       1970-01-01 00:00:00 +0000 UTC
Go version:  go1.8.3
OS/Arch:     linux/amd64
```
